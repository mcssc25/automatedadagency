Make.com Webhook Manager

Use **Make.com** (or N8N) to link your social media and ad manager accounts in minutes. Simply create a **Custom Webhook Trigger** in Make, paste the URL below, and we will send all post and ad payloads there automatically.

Trigger Payload Structure

Whenever you publish or launch, we send this JSON object to your webhook:

{
  "type": "social-post" | "ad-campaign",
  "platforms": ["linkedin", "twitter"],
  "content": "Text copywriting goes here...",
  "ctaLink": "https://www.realestatecrmpro.com",
  "timestamp": "2026-06-29T12:00:00Z"
}
